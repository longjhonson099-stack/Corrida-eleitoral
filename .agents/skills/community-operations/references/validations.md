# Community Operations - Validations

## Missing Moderation Handbook

### **Id**
no-mod-handbook
### **Severity**
high
### **Type**
conceptual
### **Check**
Should have documented moderation guidelines
### **Indicators**
  - No written mod guidelines
  - Rules in founder's head
  - Mods unsure how to handle situations
### **Message**
No moderation handbook exists.

Create comprehensive mod handbook with rules, examples, and escalation paths
