# Absurdist Voice - Sharp Edges

## Crossing The Line

### **Id**
crossing-the-line
### **Summary**
Absurdism that becomes actually offensive
### **Severity**
high
### **Situation**
"Edgy" content that hurts people
### **Why**
  Edge felt safe internally.
  Didn't test with outsiders.
  Overestimated permission level.
  
### **Solution**
  ## Line Management
  
  ### Where Lines Are
  
  ```
  Generally safe:
  - Self-deprecation
  - Meta-commentary
  - Surrealism
  - Dark humor about self
  
  Risky:
  - Other companies
  - Current events
  - Pop culture figures
  - Generational jokes
  
  Usually off-limits:
  - Protected groups
  - Tragedy
  - Politics
  - Religion
  ```
  
  ### Pre-Post Checklist
  
  | Check | Purpose |
  |-------|---------|
  | Diverse review | Catch blind spots |
  | Context check | Current events |
  | Worst interpretation | How could this be read? |
  | Screenshot test | Standalone meaning |
  
  ### Recovery Framework
  
  | Severity | Response |
  |----------|----------|
  | Minor misread | Clarify once, move on |
  | Moderate offense | Acknowledge, apologize |
  | Serious harm | Full public apology + action |
  
  ### Building Radar
  
  ```
  Develop sensitivity over time:
  
  1. Study past brand failures
  2. Diverse team input
  3. Community feedback loops
  4. Regular edge recalibration
  ```
  
### **Symptoms**
  - Unexpected backlash
  - This isn't funny
  - Trending for wrong reasons
### **Detection Pattern**
offensive|too far|apologize

## Character Drift

### **Id**
character-drift
### **Summary**
Absurdist character becoming inconsistent
### **Severity**
medium
### **Situation**
Brand voice varies wildly between posts
### **Why**
  Multiple people creating.
  No character guide.
  Trend-chasing overrode character.
  
### **Solution**
  ## Character Consistency
  
  ### Character Bible
  
  ```
  Document the character:
  
  CORE IDENTITY
  - Who is this character?
  - What do they want?
  - How do they see the world?
  
  VOICE
  - How do they speak?
  - What words do they use?
  - What's their energy?
  
  BOUNDARIES
  - What would they never say?
  - What topics do they avoid?
  - What's their line?
  ```
  
  ### Content Validation
  
  | Check | Question |
  |-------|----------|
  | Voice | "Does this sound like them?" |
  | Action | "Would they do this?" |
  | Topic | "Would they care about this?" |
  | Tone | "Is the energy right?" |
  
  ### Team Alignment
  
  ```
  Keep team consistent:
  
  1. Onboarding includes character training
  2. Regular voice calibration sessions
  3. Examples of on/off voice
  4. Central voice owner
  ```
  
  ### Evolution vs Drift
  
  | Evolution (Good) | Drift (Bad) |
  |------------------|-------------|
  | Intentional growth | Accidental change |
  | Core identity intact | Core shifts |
  | Announced changes | Confused changes |
  
### **Symptoms**
  - This doesn't sound like you
  - Engagement varies wildly
  - Brand feels fragmented
### **Detection Pattern**
inconsistent|different|which voice

## Leadership Panic

### **Id**
leadership-panic
### **Summary**
Management pulling back after success
### **Severity**
medium
### **Situation**
Viral success triggers fear
### **Why**
  Attention is scary.
  Success brings scrutiny.
  Fear of going too far.
  
### **Solution**
  ## Managing Upward
  
  ### Pre-Success Preparation
  
  ```
  Before going viral:
  
  1. Document strategy rationale
  2. Pre-approve escalation ladder
  3. Set metrics expectations
  4. Define success criteria
  5. Create risk framework
  ```
  
  ### Post-Viral Conversation
  
  | Concern | Response |
  |---------|----------|
  | "Too much attention" | "This is working" |
  | "What if backlash?" | "Here's our protocol" |
  | "Tone it down" | "Here's the data" |
  | "Stay professional" | "This IS the strategy" |
  
  ### Data Defense
  
  ```
  Metrics to track:
  
  - Engagement comparison
  - Sentiment analysis
  - Share of voice
  - Brand awareness
  - Sales correlation
  
  Let data support the weird.
  ```
  
  ### Graduated Escalation
  
  | Success Level | Leadership Update |
  |---------------|-------------------|
  | Normal engagement | Regular report |
  | 5x engagement | Heads up |
  | Viral moment | Proactive briefing |
  | Trending | Exec awareness |
  
### **Symptoms**
  - Let's be more careful
  - Maybe tone it down
  - Voice gets diluted
### **Detection Pattern**
conservative|careful|professional

## Absurdism Fatigue

### **Id**
absurdism-fatigue
### **Summary**
Audience tired of constant chaos
### **Severity**
medium
### **Situation**
What was fresh becomes exhausting
### **Why**
  No variation in energy.
  Always at 11.
  Novelty wore off.
  
### **Solution**
  ## Energy Management
  
  ### Absurdism Rhythm
  
  ```
  Not every post needs to be unhinged:
  
  Peak chaos: 20% of content
  Moderate weird: 40% of content
  Grounded weird: 40% of content
  
  Contrast makes peaks stand out.
  ```
  
  ### Content Mix
  
  | Type | Purpose |
  |------|---------|
  | Peak unhinged | Viral moments |
  | Character moments | Relationship building |
  | Actually useful | Value delivery |
  | Meta-commentary | Self-awareness |
  
  ### Refreshing the Formula
  
  ```
  Keep it fresh:
  
  1. Evolve character over time
  2. New formats for same voice
  3. Seasonal variations
  4. Surprise sincere moments
  ```
  
  ### Fatigue Signals
  
  | Signal | Meaning |
  |--------|---------|
  | Declining engagement | Formula tired |
  | "Same old" comments | Predictable |
  | Parody accounts | Peak reached |
  
### **Symptoms**
  - Diminishing returns on weird
  - They're always like this
  - Engagement plateau
### **Detection Pattern**
always the same|predictable|tiresome
