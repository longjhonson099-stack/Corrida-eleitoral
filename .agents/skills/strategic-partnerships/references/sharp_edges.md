# Strategic Partnerships - Sharp Edges

## Asymmetric Partnerships

### **Id**
asymmetric-partnerships
### **Summary**
Partnering with companies much larger or smaller
### **Severity**
high
### **Situation**
Significant size mismatch between partners
### **Why**
  Priorities differ.
  Power imbalance.
  Attention mismatch.
  
### **Solution**
  ## Managing Size Asymmetry
  
  ### When You're Smaller
  
  | Challenge | Mitigation |
  |-----------|------------|
  | Low priority | Find champion who cares |
  | Slow process | Patient persistence |
  | Changing contacts | Build multiple relationships |
  | Unfavorable terms | Know your walkaway |
  
  ```
  Strategies:
  - Find the team that needs YOU
  - Make their champion successful
  - Document everything
  - Build internal advocates
  - Be the easy partner
  ```
  
  ### When You're Larger
  
  | Challenge | Mitigation |
  |-----------|------------|
  | Dependency risk | Diversify partner base |
  | Quality variance | Clear standards |
  | Support drain | Scalable enablement |
  | Over-promising | Realistic expectations |
  
  ### Right-Sizing
  
  ```
  Questions to ask:
  
  1. Are we a priority for them?
  2. Are they a priority for us?
  3. Can we get executive attention?
  4. Is there mutual dependency?
  
  If no to most → reconsider partnership.
  ```
  
### **Symptoms**
  - Unanswered emails
  - Delayed decisions
  - Unfavorable terms
### **Detection Pattern**
enterprise partner|big company|startup partner

## Partnership Distraction

### **Id**
partnership-distraction
### **Summary**
Partnerships consuming focus without results
### **Severity**
medium
### **Situation**
Significant investment, unclear returns
### **Why**
  Partnerships feel productive.
  Sunk cost bias.
  Hope over evidence.
  
### **Solution**
  ## Avoiding Partnership Distraction
  
  ### Time Audit
  
  ```
  Track actual investment:
  
  | Activity | Hours/Week |
  |----------|------------|
  | Partner calls | X |
  | Content creation | X |
  | Integration work | X |
  | Support | X |
  | Travel | X |
  | Total | X |
  
  Compare to: direct sales, product, marketing
  ```
  
  ### Value Test
  
  | Question | Check |
  |----------|-------|
  | Revenue this quarter? | $X |
  | Deals in pipeline? | N |
  | Strategic milestone? | Y/N |
  | Would we miss them? | Y/N |
  
  ### Decision Framework
  
  ```
  If partnership value unclear after 6 months:
  
  Option A: Double down (if strategic value clear)
  Option B: Reduce investment (if tactical value only)
  Option C: Exit gracefully (if value unlikely)
  
  Default to Option C unless strong evidence.
  ```
  
  ### Opportunity Cost
  
  ```
  For every hour in partnerships:
  - What else could we do?
  - What's the expected value?
  - Is this the best use?
  
  Partnerships must beat alternatives.
  ```
  
### **Symptoms**
  - Lots of activity, little revenue
  - Can't articulate value
  - Team burnout on partners
### **Detection Pattern**
spending time|partner meetings|worth it

## Dependency Risk

### **Id**
dependency-risk
### **Summary**
Over-reliance on single partner
### **Severity**
high
### **Situation**
One partner represents significant business
### **Why**
  Felt efficient.
  Relationship was strong.
  Diversification delayed.
  
### **Solution**
  ## Managing Partner Dependency
  
  ### Concentration Limits
  
  | Metric | Yellow | Red |
  |--------|--------|-----|
  | Revenue from partner | 20% | 30%+ |
  | Leads from partner | 30% | 50%+ |
  | Distribution via partner | 40% | 60%+ |
  
  ### Diversification Strategy
  
  ```
  Reduce dependency:
  
  1. Cap growth with partner
  2. Invest in alternatives
  3. Build direct channels
  4. Negotiate better terms
  
  Never let one partner control your destiny.
  ```
  
  ### Relationship Protection
  
  | Action | Purpose |
  |--------|---------|
  | Multiple contacts | Survive departures |
  | Executive relationships | Escalation path |
  | Contract terms | Legal protection |
  | Transition planning | Exit readiness |
  
  ### Early Warnings
  
  ```
  Monitor for:
  - Partner strategy shifts
  - Key contact departures
  - Competitive product hints
  - Reduced engagement
  - Changing terms
  
  React before crisis.
  ```
  
### **Symptoms**
  - One partner dominates revenue
  - Anxiety about partner changes
  - Weak alternatives
### **Detection Pattern**
main partner|biggest partner|relying on

## Cultural Mismatch

### **Id**
cultural-mismatch
### **Summary**
Partnering despite incompatible cultures
### **Severity**
medium
### **Situation**
Strategic fit but cultural clash
### **Why**
  Focused on deal not relationship.
  Assumed professionalism would prevail.
  Underestimated cultural factors.
  
### **Solution**
  ## Navigating Cultural Differences
  
  ### Cultural Assessment
  
  | Dimension | Them | Us | Gap |
  |-----------|------|-----|-----|
  | Speed | Fast/Slow | | |
  | Formality | High/Low | | |
  | Risk tolerance | High/Low | | |
  | Decision making | Consensus/Top-down | | |
  | Communication | Direct/Indirect | | |
  
  ### Compatibility Signals
  
  ```
  Green flags:
  - Similar response times
  - Aligned on quality
  - Compatible decision pace
  - Mutual respect
  
  Red flags:
  - Ignored emails
  - Different integrity standards
  - Blame culture
  - Transactional mindset
  ```
  
  ### Bridging Differences
  
  | Gap | Approach |
  |-----|----------|
  | Speed | Set explicit timelines |
  | Formality | Match their style |
  | Communication | Over-communicate |
  | Decision making | Understand their process |
  
  ### When to Exit
  
  ```
  Cultural mismatch may be unfixable if:
  
  - Fundamental values conflict
  - Trust has eroded
  - Repeated issues despite effort
  - Team dreads interactions
  
  Exit gracefully, preserve relationships.
  ```
  
### **Symptoms**
  - Frustrating interactions
  - Misunderstandings
  - Team complaints
### **Detection Pattern**
difficult partner|culture|working with them
