# Sales - Validations

## CRM Setup Checklist

### **Id**
crm-setup-checklist
### **Type**
checklist
### **Description**
Verify CRM is properly configured
### **Items**
  - Pipeline stages defined and documented
  - Required fields at each stage set
  - Lead scoring configured
  - Activity types defined
  - Email integration connected
  - Calendar integration connected
  - Team territories/ownership rules set
  - Dashboards for key metrics created
  - Automation workflows documented
  - Data backup/export process set

## Cold Email Infrastructure Setup

### **Id**
cold-email-setup-checklist
### **Type**
checklist
### **Description**
Before sending any cold emails
### **Items**
  - Separate domain purchased (not main domain)
  - Domain authenticated (SPF, DKIM, DMARC)
  - Mailboxes created and warmed up (2+ weeks)
  - Sending tool connected
  - Unsubscribe link configured
  - Physical address in signature
  - Bounce handling configured
  - Deliverability monitoring set up
  - Daily send limits configured
  - Test emails sent to spam checkers

## List Quality Check

### **Id**
list-quality-check
### **Type**
checklist
### **Description**
Before any outreach campaign
### **Items**
  - Bounce rate expected <3%
  - Emails verified by tool
  - No role-based emails (info@, sales@)
  - Recent data (<6 months)
  - Duplicates removed
  - Existing customers excluded
  - Current prospects excluded
  - Opted-out contacts excluded
  - ICP fit verified
  - Size appropriate for campaign

## Email Content Quality

### **Id**
email-quality-check
### **Type**
checklist
### **Description**
Before sending email copy
### **Items**
  - Subject line under 50 characters
  - No spam trigger words
  - Personalization tokens work
  - Clear value proposition
  - Single, clear CTA
  - Under 150 words
  - No broken links
  - Mobile-friendly format
  - Signature complete
  - Sounds human, not robotic
