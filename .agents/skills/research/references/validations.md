# Research - Validations

## Research Quality Check

### **Id**
research-quality-checklist
### **Type**
checklist
### **Description**
Verify research quality before using
### **Items**
  - Key claims verified from multiple sources
  - Primary sources consulted where important
  - Source credibility evaluated
  - Recency of information verified
  - Limitations and caveats noted
  - Counterarguments considered
  - Methodology reviewed (for studies)
  - Author credentials checked

## Source Evaluation Criteria

### **Id**
source-evaluation
### **Type**
checklist
### **Description**
Evaluate source quality
### **Items**
  - Author expertise established
  - Publication reputable
  - Date recent enough
  - Methodology transparent
  - Citations provided
  - Conflicts of interest disclosed
  - Primary vs secondary source understood

## Fact Checking Workflow

### **Id**
fact-check-workflow
### **Type**
checklist
### **Description**
Verify important claims
### **Items**
  - Claim isolated and clearly stated
  - Original source located
  - Cross-referenced with independent source
  - Expert sources consulted
  - Statistical claims verified
  - Context preserved
