# Concept Art Specialist

## Patterns


---
  #### **Name**
50-Thumbnail Exploration Protocol
  #### **Description**
    Never refine a single idea. Generate 50+ small thumbnails (2-3 inches)
    exploring wildly different directions before selecting 3-5 to develop.
    This prevents design fixation and ensures you've explored the solution space.
    
  #### **When**
Starting any design exploration
  #### **How**
    1. Set timer for 60-90 minutes
    2. Draw thumbnails at 2-3 inch size maximum
    3. Spend 1-2 minutes per thumbnail - no more
    4. Push extremes: biggest, smallest, weirdest, safest
    5. Number every thumbnail for easy reference
    6. Review only AFTER the session, not during
    7. Select top 5 based on: clarity, interest, buildability
    8. Present all 50 to art director, highlight selections
    
  #### **Examples**
    - Environment: 50 compositions for the same space
    - Character: 50 silhouettes before any detail
    - Prop: 50 functional variations of the same object

---
  #### **Name**
Value-First Composition
  #### **Description**
    Establish your image in 3 values (black, mid, white) before any color.
    This ensures readability at any size and creates clear visual hierarchy.
    Learned from Feng Zhu's industrial approach to concept efficiency.
    
  #### **When**
Developing any illustration or keyframe
  #### **How**
    1. Create a 3-value notan study first (5 min)
    2. Squint test: does the composition read?
    3. Value grouping: 70% one value, 25% second, 5% accent
    4. Place your focal point in the highest contrast area
    5. Use value to create depth (atmospheric perspective)
    6. Only add color once values are locked
    
  #### **Anti Patterns**
    - Jumping to color before establishing value structure
    - Even distribution of lights, mids, and darks
    - Focal point in low-contrast area

---
  #### **Name**
Color Script for Narrative
  #### **Description**
    Design color progressions that support story beats across sequences.
    Each keyframe's palette should feel inevitable when seen in sequence.
    Used in film (Pixar, DreamWorks) and narrative games.
    
  #### **When**
Planning visual arc for levels, sequences, or stories
  #### **How**
    1. Map story beats to emotional tones
    2. Assign color temperatures: warm=comfort, cool=tension (or subvert intentionally)
    3. Create small color chips for each major beat
    4. Ensure adjacent scenes have clear transitions
    5. Build a master color script panel showing full progression
    6. Validate: does someone unfamiliar read the emotional arc?
    

---
  #### **Name**
Iterative Refinement Pipeline
  #### **Description**
    Move from rough to refined in defined stages, getting approval at each gate.
    This prevents wasted polish on rejected directions.
    
  #### **When**
Developing concepts from exploration to final
  #### **How**
    Stage 1: Thumbnails (50+, 1-2 min each)
      → Gate: Select 3-5 directions
    Stage 2: Rough Sketches (30 min each)
      → Gate: Select 1-2 directions
    Stage 3: Tight Sketch (2-4 hours)
      → Gate: Lock design, get written approval
    Stage 4: Color Rough (1-2 hours)
      → Gate: Approve palette and mood
    Stage 5: Final Render (8-40 hours depending on deliverable)
      → Gate: Final approval with callouts
    

---
  #### **Name**
Production-Ready Callout Sheets
  #### **Description**
    Final concepts must include all information needed for 3D artists,
    animators, and technical artists to build the asset.
    
  #### **When**
Delivering final approved concepts
  #### **How**
    1. Main hero image (3/4 view for characters, establishing shot for environments)
    2. Orthographic views: front, side, back, top (as needed)
    3. Material callouts with labeled arrows
    4. Scale reference (human silhouette or known object)
    5. Color palette chips with hex values
    6. Detail breakouts for complex areas
    7. Notes on functionality, moving parts, emission
    8. Reference links for materials/textures
    

---
  #### **Name**
Ethical Photobashing Workflow
  #### **Description**
    Use photo reference to accelerate ideation and achieve realism,
    while maintaining original design and avoiding copyright issues.
    
  #### **When**
Creating realistic concepts efficiently
  #### **How**
    1. Use only licensed stock, CC0, or your own photos
    2. Photobash in early exploration phases for speed
    3. Paint over significantly - final should be 70%+ painted
    4. Transform all photos: flip, distort, recolor, composite
    5. Never use recognizable elements from copyrighted work
    6. Build a personal photo library from location shoots
    7. Document sources for legal/production records
    
  #### **Sources**
    - Unsplash, Pexels (CC0 commercial use)
    - Your own location photography
    - 3D renders from personal kitbash libraries
    - Purchased stock from Adobe Stock, Shutterstock

---
  #### **Name**
Dynamic Symmetry Composition
  #### **Description**
    Use root rectangles and their diagonals to create dynamic but balanced
    compositions. More sophisticated than rule of thirds.
    
  #### **When**
Creating keyframes and hero illustrations
  #### **How**
    1. Determine your canvas ratio (16:9, 2.35:1, etc.)
    2. Draw the baroque diagonal (corner to corner)
    3. Draw reciprocal diagonals (perpendicular to baroque)
    4. Major sinister diagonal creates opposing tension
    5. Place key elements along these armature lines
    6. Eyes, focal points at intersection nodes
    7. Use for camera framing guidance in cinematics
    

---
  #### **Name**
Design Language System
  #### **Description**
    Establish consistent visual vocabulary across all concepts for a project.
    This creates cohesion and speeds up future design decisions.
    
  #### **When**
Starting a new project or IP
  #### **How**
    1. Define shape language: angular=aggressive, curved=organic
    2. Create material palette: 5-7 core materials max
    3. Establish color rules: faction colors, environmental palettes
    4. Set proportion standards: head-to-body ratios, scale relationships
    5. Document in a living "Visual Bible" document
    6. All subsequent concepts must reference the bible
    7. Update bible when new elements are approved
    

---
  #### **Name**
Speedpainting for Ideation
  #### **Description**
    30-60 minute paintings to explore mood, lighting, and composition
    without committing to final polish. Volume over precision.
    
  #### **When**
Early visual exploration, mood studies
  #### **How**
    1. Set a hard timer - stop when it rings
    2. Large brushes only - detail brushes are banned
    3. Focus on: composition, value, mood, color temperature
    4. Accept looseness - this is ideation, not illustration
    5. Do 3-5 speedpaints before selecting direction
    6. Present as a group, let art director feel the range
    
  #### **Tips**
    - Block in big shapes first 5 minutes
    - Establish value structure by minute 15
    - Color and mood by minute 30
    - Focal point refinement last 10 minutes

---
  #### **Name**
3D-to-2D Hybrid Workflow
  #### **Description**
    Use rough 3D blockouts to establish perspective and lighting,
    then paint over for final concepts. Industry standard for efficiency.
    
  #### **When**
Complex perspectives, architectural concepts, vehicle designs
  #### **How**
    1. Block out basic forms in Blender/Maya/SketchUp (30 min max)
    2. Set up basic lighting to match intended mood
    3. Render from multiple camera angles
    4. Paint over significantly - 3D is scaffolding, not final
    5. Add organic elements, wear, story beats in paint
    6. 3D ensures accuracy; paint ensures artistry
    
  #### **Tools**
    - Blender (free, excellent for blockouts)
    - SketchUp (fast architectural blockouts)
    - Kitbash3D libraries (for complex starting points)
    - Gravity Sketch (VR for organic forms)

---
  #### **Name**
Reference Curation Protocol
  #### **Description**
    Gather and organize reference systematically before any design work.
    Reference is research, not a shortcut.
    
  #### **When**
Starting any design task
  #### **How**
    1. Gather 20-50 references BEFORE sketching
    2. Categorize: mood, color, shape, material, function
    3. Include unexpected references (nature, other industries)
    4. Create a PureRef board with clear organization
    5. Identify WHAT you're learning from each reference
    6. Never copy - synthesize multiple sources
    7. Save boards for future project onboarding
