# Development - Validations

## AI Coding Tool Setup

### **Id**
ai-tool-setup
### **Type**
checklist
### **Description**
Before using AI coding tools
### **Items**
  - Tool installed and configured
  - Authentication completed
  - Privacy settings reviewed
  - Training data opt-out (if desired)
  - Keyboard shortcuts learned
  - Context settings configured
  - Team guidelines reviewed

## Security Configuration

### **Id**
security-setup
### **Type**
checklist
### **Description**
Security setup for AI tools
### **Items**
  - No secrets in prompts policy
  - Duplication detection enabled
  - Code review process includes AI code
  - SAST scanning in place
  - API keys not in codebase
  - Team trained on AI security risks

## AI-Generated Code Review

### **Id**
ai-code-review
### **Type**
checklist
### **Description**
Review AI-generated code
### **Items**
  - Logic is correct and complete
  - Edge cases handled
  - No security vulnerabilities
  - No deprecated patterns
  - Matches project conventions
  - Tests cover new code
  - No hallucinated APIs
  - Documentation included
  - Performance acceptable
  - Complexity appropriate

## Security Review for AI Code

### **Id**
security-review
### **Type**
checklist
### **Description**
Security-specific review
### **Items**
  - No SQL injection vulnerabilities
  - No XSS vulnerabilities
  - Input validation present
  - Output encoding where needed
  - No hardcoded secrets
  - Proper authentication checks
  - Authorization verified
  - Secure randomness used
  - Crypto done correctly
  - Error messages don't leak info

## Code Quality Review

### **Id**
quality-review

checklist
### **Description**
Quality check for AI code
### **Items**

checklist
