## Color Values Must Be Specific

### **Id**
color-values-specified
### **Severity**
info
### **Type**
regex
### **Pattern**
(primary|main|dominant)\s*(color|colour)[:\s]*(red|blue|green|dark|light|bright)
### **Message**
Color specifications should use hex (#FF0000) or RGB values, not vague descriptions
### **Fix Action**
Replace vague color names with specific hex codes: 'Primary color: #E63946'
### **Applies To**
  - *.md
  - *.yaml
  - *.json

## Proportion Ratios Should Be Numeric

### **Id**
proportion-ratios-specified
### **Severity**
info
### **Type**
regex
### **Pattern**
(tall|short|wide|thin|large|small)\s*(head|body|arm|leg)
### **Message**
Proportions should be specified numerically (e.g., '6 heads tall' not 'tall body')
### **Fix Action**
Use specific ratios: 'Character is 7 heads tall with arm length of 2.5 heads'
### **Applies To**
  - *.md
  - *.yaml
