# Student Success - Sharp Edges

## No Outcome Data

### **Id**
no-outcome-data
### **Summary**
Can't prove course works
### **Severity**
high
### **Situation**
No data on student outcomes
### **Why**
  Never set up tracking.
  Didn't follow up.
  No baseline measurement.
  Only tracked completion.
  
### **Solution**
  ## Building Outcome Data
  
  ### Start Now
  - Add pre-course survey
  - Add post-course survey
  - Set up 30/90 day follow-up
  - Track specific metrics
  
  ### What to Track
  | Metric | How |
  |--------|-----|
  | Before state | Pre-survey |
  | After state | Post-survey |
  | Specific results | Follow-up survey |
  | NPS | Would you recommend? |
  
  ### Retroactive Collection
  - Email past students
  - "How are you doing 6 months later?"
  - Offer incentive for response
  - Any data is better than none
  
  ### Using Outcome Data
  - Sales page: "87% achieved X"
  - Testimonials: Specific results
  - Course improvement: What worked
  - Pricing justification: ROI story
  
### **Symptoms**
  - Can't answer "does it work?"
  - No specific result claims
  - Generic testimonials only
  - Guessing at effectiveness
### **Detection Pattern**
don't know|not sure if|no data|haven't tracked

## Testimonial Desert

### **Id**
testimonial-desert
### **Summary**
Can't get testimonials despite success
### **Severity**
medium
### **Situation**
Students succeed but don't share
### **Why**
  Didn't ask.
  Asked at wrong time.
  Made it too hard.
  No follow-up.
  
### **Solution**
  ## Getting More Testimonials
  
  ### Timing is Everything
  | Moment | Likelihood |
  |--------|------------|
  | Just completed | High |
  | Just got result | Highest |
  | Random email | Low |
  | After asking for help | Medium |
  
  ### Make It Effortless
  - 2-3 specific questions
  - 5 minutes or less
  - Option for text or video
  - Pre-fill what you can
  
  ### Follow-Up System
  ```
  Day 0: Completion celebration
  Day 7: Check in, soft ask
  Day 30: Results check, testimonial ask
  Day 90: Case study opportunity
  ```
  
  ### Incentives (Careful)
  - Discount on next course
  - Feature on website
  - Access to bonus content
  - Don't pay for testimonials (inauthentic)
  
  ### If Still No Response
  - Personal outreach to top students
  - Phone/video call to collect
  - Social media shoutouts (screenshot)
  - Wait for natural success moments
  
### **Symptoms**
  - Few testimonials despite enrollments
  - Generic, weak quotes
  - No video testimonials
  - Same old testimonials for years
### **Detection Pattern**
won't share|no testimonials|need more|haven't asked

## Abandoned Alumni

### **Id**
abandoned-alumni
### **Summary**
Alumni community dies after launch
### **Severity**
medium
### **Situation**
Alumni program has no engagement
### **Why**
  Set up and forgot.
  No ongoing value.
  No events or updates.
  Students moved on.
  
### **Solution**
  ## Reviving Alumni Community
  
  ### Quick Wins
  - Host an alumni event
  - Send value-packed newsletter
  - Share success stories
  - Create job/opportunity board
  
  ### Sustainable Model
  | Activity | Frequency |
  |----------|-----------|
  | Newsletter | Monthly |
  | Success spotlight | Bi-weekly |
  | Alumni call | Quarterly |
  | Big event | Annually |
  
  ### Value Exchange
  | You Provide | Alumni Provide |
  |-------------|----------------|
  | Network | Testimonials |
  | Updates | Referrals |
  | Events | Case studies |
  | Discounts | Community help |
  
  ### If Too Far Gone
  - Acknowledge the gap
  - Survey what they'd want
  - Relaunch with commitment
  - Start small, grow if engaged
  
### **Symptoms**
  - Dead Slack/Discord
  - No alumni engagement
  - Alumni ask "is this still active?"
  - No one shows up to events
### **Detection Pattern**
dead|inactive|no one|empty|used to be

## Certification Inflation

### **Id**
certification-inflation
### **Summary**
Certification becomes meaningless
### **Severity**
medium
### **Situation**
Everyone passes, certification has no value
### **Why**
  100% pass rate.
  No real assessment.
  Quantity over quality.
  Pressure to not fail anyone.
  
### **Solution**
  ## Meaningful Certification
  
  ### The Hard Truth
  - If everyone passes, it means nothing
  - 80-90% pass rate is healthy
  - Some should need to retake
  - Standards matter
  
  ### Rigor Spectrum
  | Level | Requirement |
  |-------|-------------|
  | Completion | Finish content |
  | Basic cert | Pass quiz (70%) |
  | Full cert | Complete project |
  | Advanced | Peer + instructor review |
  
  ### Quality Control
  - Blind grading where possible
  - Clear rubrics
  - Retake option (with limits)
  - Appeal process
  
  ### If Already Inflated
  - Grandfather existing
  - Announce new standards
  - Create "advanced" tier
  - Make new cert meaningful
  
### **Symptoms**
  - 100% pass rate
  - Employers don't recognize
  - Certificate is "participation trophy"
  - No one fails
### **Detection Pattern**
everyone passes|doesn't mean anything|easy|anyone can get
