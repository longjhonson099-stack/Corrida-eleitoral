# Meme Engineering - Validations

## Using Dead Meme Format

### **Id**
dead-format
### **Severity**
medium
### **Type**
conceptual
### **Check**
Meme format should still be relevant
### **Indicators**
  - Format peaked months ago
  - Mainstream media covered it
### **Message**
Format is past its prime.
### **Fix Action**
Use fresher format or wait for ironic revival
